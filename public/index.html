### TIPS ###
Encierra tú texto en estas etiquetas para características especiales


<p>esto es un texto normal</p>
      - Para párrafos de texto.


<b>esto se imprimirá en negrita</b>
  - Para resaltar texto en negrita.


<i>esto es cursiva</i>
              - Para aplicar cursiva a tu texto.


<h1>Encabezado principal</h1>
      - Para el encabezado más grande, el título principal.


<h2>Encabezado secundario</h2>
     - Para subtítulos o encabezados de segundo nivel.


<h3>Encabezado terciario</h3>
     - Para encabezados de tercer nivel.


<ul><li>Elemento de lista</li></ul>
  - Para crear listas con viñetas.


<ol><li>Elemento numerado</li></ol>
 - Para listas ordenadas.


<code>Código</code>
              - Para mostrar fragmentos de código.


<hr>
                                - Para crear una línea horizontal separadora.
